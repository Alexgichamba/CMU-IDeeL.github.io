\bf{Forward Pass} \linebreak Input: $D$ dimensional vector $\mathbf{x} = [x_j, j=1..D]$ \linebreak Set: \linebreak $\quad $ $D_0=0,$ is the width of the $0^\text{th}$ (input) layer \linebreak $\quad $ $y_j^{(0)}=x_j,$ $j=1..D$ \linebreak $\quad $ $y_0^{(k=1..N)}=x_0=1$ \linebreak For layer $k = 1..N$ \linebreak $\quad $ For $j=1..D_k$ \linebreak $\qquad$ $z_j^{(k)}=\sum_{i=0}^{N_k}w_{i,j}^{(k)}y_i^{(k-1)}$ \linebreak $\qquad$ $y_j^{(k)}=f_k(z_j^{(k)})$ \linebreak Output: \linebreak $\quad $ $Y=y_j^{(N)},$ $j=1..D_N$ \linebreak
